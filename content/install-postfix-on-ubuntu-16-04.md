Postfix
-------

Install postfix:

`sudo apt install postfix`

Accept all default settings, confirming configure as _internet site_ and double check your [FQDN](https://en.wikipedia.org/wiki/Fully_qualified_domain_name) ( ie: mail.example.com ).
